Installation

Goo runs within Blender, so you will first need to download Blender 3.6 LTS from here. The package was developed under Blender 3.6 LTS, so we recommend using this version. We aim to maintain Goo for future Blender LTS versions but might have a lag form their releases.

Once you have Blender installed:

  1. Download the Goo library from GitHub.

  2. In Blender, Edit > Preferences > File Paths > Scripts: add <your_root_path>/Goo/scripts

Simulation scripts

Examples of simulation script can be found in the /simulations folder, located here. Once you get a good grasp of the library, you will be able to write your own Goo scripts and specify lots of initial conditions for your simulations of cells.

Goo scripts typically get ran from Blender’s scripting tab, though they can be ran from Visual Studio Code directly using the developer’s extension developed by Jacques Lucke.

Dependencies

Goo has the following dependencies.

  • python 3.7 or newer

  • numpy

  • matplotlib

  • bpy

  • bmesh

  • mathutils
